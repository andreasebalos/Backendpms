## Quick Start
1. Install dependencies:
```bash
pnpm install
```

2. Set up environment:
```bash
cp .env.example .env
```

3. Start development server:
```bash
pnpm dev
```

4. Run tests:
```bash
pnpm test
```

- Product Management
